# Commit message


| 태그       | 제목 설명                                             |
|------------|------------------------------------------------------|
| Feat       | 새로운 기능 추가                                     |
| Fix        | 버그 수정                                            |
| Docs       | 문서 수정                                            |
| Style      | 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우    |
| Refactor   | 코드 리팩토링                                        |
| Test       | 테스트 코드 추가, 수정, 삭제 (비즈니스 로직 변경 없음)|
| Chore      | 기타 변경사항 (빌드, 이미지, 패키지 등)              |
| Design     | CSS 등 사용자 UI 디자인 변경                         |
| Comment    | 필요한 주석 추가 및 변경                             |
| Init       | 프로젝트 초기 생성                                   |
| Rename     | 파일 혹은 폴더명 수정 또는 이동                      |
| Remove     | 파일 삭제만 수행하는 경우                             |
